## 세션 — 계약 탭 개선 (집에서 작업, ~v4.13.1)

### 환경 (집 PC에서 새로 세팅함)
- Node.js LTS 설치, `npm install` 완료
- git 연결됨 (`origin/main` 추적). 작업 흐름: **작업 시작 전 `git pull` → 작업 → 커밋 → push** (main 직접, Vercel 자동 배포)
- **커밋 컨벤션·로컬 실행법은 [CLAUDE.md](CLAUDE.md) 참고** (이번에 문서화함)
- 로컬 실행:
  ```
  npm run dev                  # 프론트 5173
  vercel dev --listen 5601     # 백엔드(계약/행정 API) 5601
  ```
- 계약 탭 백엔드는 `NOTION_TOKEN`이 Vercel **Development 스코프**에 있어야 로컬에서 동작 (이번에 추가함). 값은 Sensitive라 `vercel env pull`로 안 나옴 → 원본(Notion 인테그레이션 시크릿)에서 가져와야 함.
